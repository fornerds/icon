# @fornerds/icon

Fornerds 아이콘 라이브러리

## 설치

```bash
npm install @fornerds/icon
```

## 사용법

```tsx
import { ArrowRight } from '@fornerds/icon';

function App() {
  return (
    <div>
      <ArrowRight size={24} />
      <ArrowRight size={32} color="#3182f6" />
    </div>
  );
}
```

## Props

- `size`: 아이콘 크기 (기본값: 24)
- `color`: 아이콘 색상
- `className`: CSS 클래스명
- `style`: 인라인 스타일
